### How tu run
<p>Softwares needed: </p>
    <p>composer</p>
    <p>Laravel 9</p>
    <p>XAMPP</p>
    <p>VS Code</p>

<p>Install dependencies using: </p>
    <p>npm install</p>
    <p>composer update && composer instal</p>

<p>run using:</p>
    <p>php artisan migrate:fresh --seed</p>
    <p>npm run dev</p>
    <p>php artisan serv</p>
